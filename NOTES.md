NOTES:

How to create a new app:
========================

```bash
$ cd ~/create-app
$ npx create-app install.js --package=@jdeighan/create-app <new app name>
```

How to update the package
=========================

Make your changes, then:

```bash
$ cd ~/create-app
$ git add -A
$ git commit -m "<commit msg>"
$ patch
$ push
$ publish
```
